Vue.js: Framework for Web Development by Xinhui Xu

What's a framework?

A framework differs from a library, which offers various functions to be called by its parent code. A framework defines the design of an application.

Vue.js, Angular.js, Bootstrap, etc. are frameworks for building web applications. d3.js is a library with functions mainly for data-visualization.

Why Vue.js?

  • No need for external libraries
  • No need for jQuery
  • No need for installing plugins
  • Explicit & simplistic structure

*Although basic HTML/CSS/JS experience is enough to get you started, some advanced Vue syntax may require in-depth knowledge of ES5.

Get started

Include <script src="https://unpkg.com/vue"></script> in your HTML file. That's it.
